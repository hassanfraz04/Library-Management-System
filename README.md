# Library Management System

## Project Overview
This project is developed as part of the Software Project Management course. The system is designed to manage library operations including book management, user authentication, borrowing and returning books, and administrative functionalities.

## Objectives
- Automate library operations
- Improve book tracking efficiency
- Manage users and transactions
- Provide secure access for administrators and users

## Project Features
- User Authentication
- Book Management
- Borrow & Return System
- Search Functionality
- Admin Dashboard
